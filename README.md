# Taxi Booking Service

## Overview

Taxi Booking Service is a web-based application built using **PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap**. The system enables users to register, log in, check vehicle availability, and book taxis online. It is designed for users in **Vadodara**, with taxi services operating across **Gujarat**. The application also provides recommendations for popular tourist destinations within Gujarat.



## Live Demo

Website: https://chalotravels.freehosting.dev/?i=1



## Features

* User Registration and Login
* Secure Authentication
* Taxi Booking System
* Vehicle Availability Check
* Booking Confirmation
* Tourist Places Recommendation (Gujarat)
* Responsive User Interface



## Technologies Used

Frontend : HTML5, CSS3, JavaScript
Backend : PHP 
Database : MySQL
